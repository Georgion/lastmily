# Angular Test Assignment

## Start the frontend

Navigate to `lastmily/frontend/shipment-management`

Install dependencies

```bash
 npm i
```

Start a local server


```bash
 npm start
```


## Start the backend

Navigate to `lastmily/backend/mock-server`

Install dependencies

```bash
 npm i
```

Start a local server


```bash
 npm start
```
